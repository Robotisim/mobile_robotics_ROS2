## Requirements
### Software
- Ubuntu 22.04
- ROS2 Humble
- Vscode

### Hardware
- Raspberry Pi 3B+
- Arduino Uno
- 2x DC Motors
- 2x Wheels
- 1x Caster Wheel
- 1x L298N Motor Driver
- 1x 12V Battery
